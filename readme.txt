== Description ==

This is a count down widget for Dpub.

== Installation ==

* Download the .zip of this plugin.
* Upload it to the Wordpress via plugin uploader and activate the plugin.
* Layout a Dpub count down widget on WP Dashboard.
* You can change a widget title, if you want to change.
